# Motion Design Course – HTML, CSS & JavaScript Exercises

This repository contains exercises and practice files created while taking a **How to Animate Websites / Motion Design** course.

## About

The goal of this repository is to learn and experiment with the fundamentals of front-end development used in motion design and interactive web experiences.

The exercises focus on:

* Basic HTML structure
* CSS styling fundamentals
* JavaScript interactions
* Animation concepts and motion principles
* Building interactive web components

## Project Structure

```text
project/
│
├── exercise-01.html
├── exercise-02.html
├── exercise-03.html
├── exercise-04.html
└── ...
```

For simplicity and to match the course exercises:

* Each exercise is contained entirely within its own `.html` file
* HTML, CSS, and JavaScript are written together in the same file
* No external CSS or JavaScript files are used unless required later in the course

Example structure inside an exercise file:

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        /* CSS code */
    </style>
</head>

<body>

    <!-- HTML content -->

    <script>
        // JavaScript code
    </script>

</body>
</html>
```

## Topics Covered

* HTML basics
* Elements and layout structure
* CSS styling
* Positioning
* Transitions
* Transforms
* Keyframe animations
* JavaScript basics
* DOM manipulation
* Event handling
* Interactive motion effects

## Running the Exercises

Clone the repository:

```bash
git clone <repository-url>
```

Open any exercise file in your browser:

```bash
exercise-01.html
```

Or use a local development server if preferred.
